# Subnetting Lab

## Objective
Configure a multi-router network with Subnetting using Packet Tracer.

## Topology
- 4 Routers connected in full mesh
- Each Router connects to one Switch
- Each Switch connects to a single PC

## Skills
- Subnetting
- VLSM
- Router interface configuration
- Inter-router connectivity

## Network Summary
- 192.168.1.0/24 network subdivided into 4 subnets:
  - /26 subnets for LAN PCs
  - /30 subnets for router-to-router links
